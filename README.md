#  Bot de Gestión de Citas

Aplicación web para automatizar la programación y gestión de citas.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- Node.js
- MongoDB Atlas

##  Funcionalidades

- Registro de citas con fecha y hora
- Consulta de citas programadas
- Almacenamiento en base de datos en la nube
- Interfaz web interactiva

##  Cómo ejecutar el proyecto

1. Clona el repositorio
   git clone https://github.com/hawer12/bot-citas

2. Instala las dependencias
   npm install

3. Crea un archivo .env con tus credenciales
   MONGODB_URI=tu_uri_aqui
   PORT=3000

4. Inicia el servidor
   node index.js
